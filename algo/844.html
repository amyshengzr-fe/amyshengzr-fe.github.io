844. 比较含退格的字符串 ​

题目描述 ​

给定 s 和 t 两个字符串，当它们分别被输入到空白的文本编辑器后，如果两者相等，返回 true 。# 代表退格字符。

注意：如果对空文本输入退格字符，文本继续为空。

示例 1：

输入：s = "ab#c", t = "ad#c"
输出：true
解释：s 和 t 都会变成 "ac"。

示例 2：

输入：s = "ab##", t = "c#d#"
输出：true
解释：s 和 t 都会变成 ""。

示例 3：

输入：s = "a#c", t = "b"
输出：false
解释：s 会变成 "c"，但 t 仍然是 "b"。

提示：

  • 1 <= s.length, t.length <= 200
  • s 和 t 只含有小写字母以及字符 '#'

进阶：

  • 你可以用 O(n) 的时间复杂度和 O(1) 的空间复杂度解决该问题吗？

题解 ​

代码 ​

js
const backspace = function (val) {
  let arr = [];
  for (let i = 0; i < val.length; i++) {
    if (val[i] === "#") {
      arr.pop();
    } else {
      arr.push(val[i]);
    }
  }
  return arr.join("");
};

var backspaceCompare = function (s, t) {
  return backspace(s) === backspace(t);
};

代码功能拆解分析 ​

1. backspace 函数 ​

js
const backspace = function (val) {
  let arr = [];
  for (let i = 0; i < val.length; i++) {
    if (val[i] === "#") {
      arr.pop();
    } else {
      arr.push(val[i]);
    }
  }
  return arr.join("");
};
核心思想 ​

  • 使用 栈（Stack） 的思想来模拟字符串输入和退格过程

  • arr 作为辅助栈：

    • 普通字符 → 入栈
    • # → 出栈（模拟删除前一个字符）
执行流程示例 ​

以 "ab#c" 为例：

字符操作栈内容
apush[a]
bpush[a, b]
#pop[a]
cpush[a, c]

最终返回 "ac"

边界情况 ​
  • 当栈为空时遇到 #，pop() 返回 undefined，不会报错，逻辑安全

2. backspaceCompare 函数 ​

js
var backspaceCompare = function (s, t) {
  return backspace(s) === backspace(t);
};
逻辑说明 ​
  • 分别对 s 和 t 进行退格处理
  • 比较处理后的最终字符串是否相等
  • 返回布尔值结果

整体算法思路总结（栈模拟） ​

  1. 定义一个辅助栈

  2. 遍历字符串：

    • 普通字符入栈
    • # 执行退格（出栈）

  3. 将栈中剩余字符拼接为最终字符串

  4. 对两个字符串分别执行上述过程

  5. 比较最终结果

时间与空间复杂度分析 ​

时间复杂度 ​

  • backspace：O(n)
  • backspaceCompare：O(n + m)

其中 n 和 m 分别是字符串 s 和 t 的长度

空间复杂度 ​

  • O(n)
  • 使用了额外的栈来存储字符

进阶 ​

  • 双指针、O(1) 额外空间复杂度

核心思想（从后向前遍历） ​

退格符 # 的影响只作用于它左侧最近的未删除字符，因此可以：

  • 从字符串末尾开始遍历
  • 使用计数器记录尚未生效的退格次数
  • 跳过所有被退格删除的字符
  • 逐个比较两个字符串中“有效字符”

整个过程不需要额外栈结构，空间复杂度为 O(1)。

算法步骤说明 ​

对字符串 s 和 t：

  1. 分别设置指针 i、j 指向字符串末尾

  2. 设置退格计数器 skipS、skipT

  3. 移动指针以找到下一个有效字符：

    • 遇到 #，退格计数加一

    • 遇到普通字符：

      • 若退格计数 > 0，则该字符被删除，计数减一
      • 否则，该字符为有效字符

  4. 比较当前有效字符：

    • 不相等直接返回 false

  5. 同步左移指针，继续比较

  6. 全部遍历完成则返回 true

代码实现 ​

js
var backspaceCompare = function (s, t) {
  let i = s.length - 1;
  let j = t.length - 1;

  let skipS = 0;
  let skipT = 0;

  while (i >= 0 || j >= 0) {
    // 处理 s
    while (i >= 0) {
      if (s[i] === "#") {
        skipS++;
        i--;
      } else if (skipS > 0) {
        skipS--;
        i--;
      } else {
        break;
      }
    }

    // 处理 t
    while (j >= 0) {
      if (t[j] === "#") {
        skipT++;
        j--;
      } else if (skipT > 0) {
        skipT--;
        j--;
      } else {
        break;
      }
    }

    // 比较当前有效字符
    const charS = i >= 0 ? s[i] : null;
    const charT = j >= 0 ? t[j] : null;

    if (charS !== charT) {
      return false;
    }

    i--;
    j--;
  }

  return true;
};

示例执行过程 ​

以：

s = "ab#c"
t = "ad#c"

从后向前：

  • 比较 c == c
  • # 出现，跳过 b 和 d
  • 比较 a == a
  • 全部一致 → 返回 true

复杂度分析 ​

指标复杂度
时间复杂度O(n + m)
空间复杂度O(1)

仅使用常数级变量，不依赖额外数据结构

与栈解法对比 ​

方案空间复杂度可读性是否最优
栈模拟O(n)高否
双指针O(1)中是

通过从后向前遍历字符串并记录退格数量，可以在不构造新字符串的情况下逐字符比较两个字符串的有效内容，从而将空间复杂度优化至 O(1)。
