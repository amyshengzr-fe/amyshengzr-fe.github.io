367. 有效的完全平方数 ​

题目描述 ​

给你一个正整数 num 。如果 num 是一个完全平方数，则返回 true ，否则返回 false 。

完全平方数 是一个可以写成某个整数的平方的整数。换句话说，它可以写成某个整数和自身的乘积。

不能使用任何内置的库函数，如 sqrt 。

示例 1：

输入：num = 16
输出：true
解释：返回 true ，因为 4 * 4 = 16 且 4 是一个整数。

示例 2：

输入：num = 14
输出：false
解释：返回 false ，因为 3.742 * 3.742 = 14 但 3.742 不是一个整数。

提示：

  • 1 <= num <= 2³¹ - 1

题解 ​

代码 ​

js
var isPerfectSquare = function (num) {
  let left = 0,
    right = num;
  while (left <= right) {
    const mid = (left + right) >> 1;
    const y = mid * mid;
    if (y === num) {
      return true;
    } else if (y > num) {
      right = mid - 1;
    } else {
      left = mid + 1;
    }
  }

  return false;
};

    3. 如果 mid*mid < num → 中点平方太小 → 搜索右半区 left = mid + 1。

  • 如果循环结束都没找到平方等于 num，说明 num 不是完全平方数 → 返回 false。

代码特点 ​

  1. 二分查找：

    • 时间复杂度：O(log n)，比直接遍历到 sqrt(num) 更高效。
    • 空间复杂度：O(1)，没有使用额外数组或递归。

  2. 防溢出注意：

    • 在 JavaScript 中，数字是 64 位浮点数，一般不会因为 mid * mid 溢出。

  3. 效率优化：

    • 使用 >> 1 替代 /2，略微提高整数除法效率。

示例 ​

  1. 输入：num = 16

    • 二分查找过程：

      • left=0, right=16 → mid=8 → 8*8=64 > 16 → right=7
      • mid=3 → 3*3=9 < 16 → left=4
      • mid=5 → 5*5=25 > 16 → right=4
      • mid=4 → 4*4=16 = 16 → return true

  2. 输入：num = 14

    • 最终循环结束没有找到平方等于 14 → return false

题解思路总结 ​

题目本质：判断一个数是否是完全平方数 方法选择：二分查找（推荐） 步骤：

  1. 左右边界 [0, num]

  2. 循环计算中点平方：

    • 等于 → 返回 true
    • 大于 → 搜索左边
    • 小于 → 搜索右边

  3. 循环结束返回 false

复杂度分析：

  • 时间复杂度：O(log n)
  • 空间复杂度：O(1)
