阿珍笔记 ​

欢迎来到 阿珍笔记，这是一个使用 VitePress 构建的个人技术博客和学习笔记网站。

📝 项目介绍 ​

本项目是一个个人知识库和技术博客，旨在记录项目开发经验、分享技术知识和解决问题的思路。正如站点副标题所言："好记性不如烂笔头"，通过持续记录和整理，形成系统化的知识沉淀。

✨ 主要功能 ​

1. 项目开发记录 ​

  • 从个人产品到学习探索的完整记录
  • 包含产品需求文档、开发过程、API 设计和系统架构
  • 示例项目：We-Dish（一个美食 AI 推荐系统）

2. 知识分享 ​

  • 汇总精选工具使用技巧
  • 最佳实践分享
  • 解决记录与复盘

3. 技术文档 ​

  • 详细的项目文档和接口定义
  • 开发指南和配置说明

🛠 技术栈 ​

  • 框架: VitePress 2.0.0-alpha.12
  • 语言: TypeScript 5
  • 前端: Vue 3
  • 构建工具: Vite

🚀 快速开始 ​

开发环境 ​

bash
nvm -v 0.39.1
node -v v22.19.0
pnpm -v 10.15.1

安装依赖 ​

bash
# 使用 pnpm（推荐）
pnpm install

# 或使用 npm
npm install

# 或使用 yarn
yarn install

开发环境运行 ​

bash
pnpm docs:dev
# 或
npm run docs:dev
# 或
yarn docs:dev

然后访问 http://localhost:5173 查看网站。

构建静态网站 ​

bash
pnpm docs:build
# 或
npm run docs:build
# 或
yarn docs:build

构建后的文件将位于 .vitepress/dist 目录。

预览构建结果 ​

bash
pnpm docs:preview
# 或
npm run docs:preview
# 或
yarn docs:preview

📁 项目结构 ​

├── .github/          # GitHub 工作流配置
├── .vitepress/       # VitePress 配置和主题
│   ├── config.mts    # 站点配置文件
│   └── theme/        # 自定义主题
├── proj/             # 项目文档
│   └── we-dish/      # We-Dish 项目文档
├── public/           # 静态资源
├── share/            # 知识分享和工具使用指南
└── index.md          # 网站首页

🔧 自定义配置 ​

主要配置文件位于 .vitepress/config.mts，可以在这里修改：

  • 站点标题、描述和语言
  • 导航栏和侧边栏结构
  • 搜索功能设置
  • 主题颜色和字体
  • 页脚信息

📤 部署 ​

本项目配置了 GitHub Actions 自动部署工作流，位于 .github/workflows/deploy.yml。当推送到主分支时，将自动构建并部署到 GitHub Pages。
