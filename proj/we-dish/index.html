We Dish 项目总览 ​

AI 智能菜品推荐与菜品订单管理系统，现支持 C 端用户版和 B 端管理版 自适应 web 平台。

1.项目概述 ​

We-Dish 是一款专注于美食管理的自适应 Web 应用，旨在帮助用户发现、管理和下单菜品。通过智能推荐、便捷的菜单浏览和订单管理功能，为用户提供优质的美食体验。

1.1 核心价值 ​

  • 智能推荐：通过 AI 聊天功能为用户提供个性化菜品建议
  • 便捷管理：用户可以轻松浏览菜单、管理购物车和查看历史订单
  • 个性化设置：支持用户自定义个人信息和账户安全设置

1.2 目标用户 ​

  • 美食爱好者
  • 需要管理菜品和订单的餐厅用户
  • 寻求菜品推荐的用户群体

1.3 项目特点 ​

  • 📚 完整文档 - 从产品需求到开发指南，一应俱全
  • 🏗️ 清晰架构 - 分离用户版和管理版，各自独立开发
  • 📦 版本管理 - 支持多个版本的并行维护
  • 🎨 现代技术栈 - 使用最新的前端框架和最佳实践

2.技术栈 ​

开发环境

nvm -v 0.39.1
node -v v22.19.0
pnpm -v 10.15.1

2.1 用户版（Lite） ​

  • 框架: Next.js 15（App Router） + React 18
  • 状态管理: Redux Toolkit 9.x
  • UI 组件库: Ant Design 5.x
  • 样式方案: Tailwind CSS 4.x
  • 开发语言: TypeScript 5.x
  • HTTP: axios
  • 工具库: js-cookie（鉴权）, ahooks, clsx, lucide-react

2.2 管理版（Admin） ​

  • 框架：Vue 3 + Vite 7
  • 路由管理：Vue Router 4.x
  • UI 库：Element Plus 2.x
  • 状态管理：Pinia 3.x
  • HTTP：axios
  • 语言：TypeScript 5.x
  • 样式：Unocss 66.x
  • 工具库：js-cookie（鉴权）、VueUse 14

3.版本迭代 ​

3.1 v1.0-基础功能版 (当前稳定版) ​

这是第一个完整功能版本，包含系统的核心功能和完整的文档。

主要功能：

  • ✅ 用户认证与授权
  • ✅ AI 对话辅助
  • ✅ 菜品管理（CRUD）
  • ✅ 菜单管理
  • ✅ 购物车管理（添加、删除、更新）
  • ✅ 订单管理（创建、查询）
  • ✅ 响应式设计
